Accessibility

A major factor and driving force when this website was in the design phase was accessibility.
It must be easy to read.
Not only for people who can see well, but by people who have vision difficulties too.
It was designed so it can be displayed in a text terminal just as well it is show in a graphical web browser.
This website should too be quite clear when read by any kind of screen reader.

In this website we have function over form.
It is shaped and coded so it works well, loads fast and fits in your screen.
