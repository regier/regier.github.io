Telegram For Linux Installer

TelegramForLinuxInstaller is a tool that you can use to automagically download the latest stable version of Telegram for Linux from Telegram.org and install it into your system.

It ships in two versions. A deb package for convenience where the user just download and install the package and a script that should work mostly on any POSIX complient OS.

My goal with this project was improving some of my shell script skills, understanding condition statements, learning and following the KISS Design Principle, where each program/script does one thing only. While at the same time, using KISS to help me achieving a nice modular design.
TFLI (TelegramForLinuxInstaller) got so modular in fact that at the very same time, without noticing I made it portable to any OS by design.
