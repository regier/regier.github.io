  • First we need to install some system stuff and dependencies
    • apt install dnsmasq-base ebtables dnsmasq-base iptables apparmor
  • Now lets install some other utils that may help us with some configuration
    • apt install libvirt-clients libvirt-daemon-system lsb-base lsb-release debian-goodies curl
  • And finally let's install LXC and the tools to create and manage our LXC containers.
    • apt install lxc lxctl lxcfs lxc-templates debootstrap
  • Now make your /etc/lxc/default.conf look like this
    • lxc.net.0.type = veth
lxc.net.0.link = lxcbr0
lxc.net.0.flags = up
lxc.apparmor.profile = generated
lxc.apparmor.allow_nesting = 1
